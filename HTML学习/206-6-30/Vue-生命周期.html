vue的生命周期:指的是vue对象从创建到销毁的过程。 vue的生命周期包含8个阶段:每触发一个生命周期事件,会自动执行一个生命周期方法,这些生命周期方法也被称为钩子方法。 | 状态 | 阶段周期 | | ------------- | -------- | | beforeCreate | 创建前 | | created | 创建后 | | beforeMount | 挂载前 | | mounted | 挂载完成 | 主要知道 | beforeUpdate | 更新前 | | updated | 更新后 | | beforeDestroy | 销毁前 | | destroyed | 销毁后 |
